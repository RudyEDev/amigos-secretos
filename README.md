# 🎁 Amigo Secreto

Aplicación web para organizar un sorteo de **Amigo Secreto**.

## 🚀 Funcionalidades
- Agregar nombres de los participantes.
- Listar amigos agregados.
- Realizar el sorteo aleatorio.
- Mostrar los resultados.
